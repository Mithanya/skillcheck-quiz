# SkillCheck – Interactive Quiz Application

## Overview

SkillCheck is a lightweight and responsive quiz application designed to test fundamental web development knowledge. It provides an intuitive user experience with real-time scoring and detailed feedback.

## Live Demo

https://mithanya.github.io/skillcheck-quiz/

## Features

* Five technical questions (HTML, CSS, JavaScript fundamentals)
* Next and Previous navigation buttons
* Answer validation – cannot proceed without a selection
* Live score tracking during the quiz
* Final results screen with question-wise feedback
* Fully responsive layout (desktop, tablet, mobile)
* Zero external dependencies – pure vanilla implementation

## Technologies Used

* HTML5
* CSS3 (Flexbox, custom styling)
* JavaScript (ES6)
* DOM manipulation

## Installation & Setup

1. Clone the repository or download the ZIP file.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3. No build steps or server required.

## How to Use

1. Read each question and select an answer.
2. Click **Next** to proceed (selection required).
3. Use **Previous** to navigate back.
4. Click **Finish** after the last question.
5. View your score and detailed feedback.
6. Restart anytime to retake the quiz.
